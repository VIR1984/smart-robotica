📱 СИНХРОНИЗАЦИЯ НА ВСЕ УСТРОЙСТВА
Почему мобильный видит старые данные?
localStorage хранится отдельно в каждом браузере на каждом устройстве. Чтобы данные появились на телефоне — нужно скачать файл cms-data.js и загрузить его на хостинг рядом с index.html.

Инструкция: 1) Нажмите кнопку ниже → скачается cms-data.js. 2) Загрузите этот файл на хостинг в корневую папку сайта (рядом с index.html). 3) Все устройства будут видеть одинаковые данные.

КАК ЭТО РАБОТАЕТ
  1. Вы редактируете данные в админке → они сохраняются в localStorage вашего браузера
  2. Нажимаете «Скачать cms-data.js» → скачивается файл с вашими данными
  3. Загружаете файл на хостинг (FTP, cPanel, Netlify drag-and-drop и т.д.)
  4. Теперь все браузеры и устройства читают данные из этого файла
  5. При следующих изменениях — повторите шаги 2-3
